# Chinese Character Learner - Website

This is the landing page website for the Chinese Character Learner Chrome extension.

## Files

- `index.html` - Main landing page
- `styles.css` - Styling for the website
- `README.md` - This file

## Customization

### Before Publishing:

1. **Update Chrome Web Store Link**
   - Replace `https://chrome.google.com/webstore` with your actual extension URL once published

2. **Add Screenshots**
   - Replace the screenshot placeholders with actual screenshots of your extension
   - Recommended sizes: 1280x800 or 640x400 pixels
   - Show: Quiz interface, Word Metrics page, Activity Grid

3. **Add Privacy Policy**
   - Create a `privacy.html` page or link to your privacy policy
   - Update the footer link

4. **Add Support Information**
   - Create a support page or link to your support channel
   - Update the footer link

5. **Customize Content**
   - Update any text to match your exact features
   - Add any additional information about the extension

## Chrome Web Store Listing

For your Chrome Web Store listing, you'll need:

1. **Description** (132+ characters, 16,000 max)
2. **Screenshots** (1280x800 or 640x400)
3. **Promotional images** (optional)
4. **Privacy policy URL**
5. **Category** (Education or Productivity)
6. **Language** (English)
