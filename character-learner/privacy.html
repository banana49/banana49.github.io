Privacy Policy

Chinese Character Learner Extension

Last Updated: 12/27/2025

Introduction

Chinese Character Learner is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our Chrome extension.

Information We Collect

Data Stored Locally

Our extension stores all data locally on your device using Chrome's local storage. This includes:

  • Chinese words and vocabulary you add
  • Pinyin transcriptions
  • Quiz performance data (correct/incorrect answers)
  • Practice session information
  • API keys (if you choose to use AI features)

We do not collect, transmit, or store any of this data on our servers.

Optional AI Features

If you choose to use the AI sentence generation feature:

  • Your OpenAI API key is stored locally on your device
  • When generating sentences, words are sent to OpenAI's API
  • We do not have access to your API key or API usage
  • Please review OpenAI's privacy policy for their data handling

How We Use Your Information

All data is used solely within the extension to:

  • Provide quiz functionality
  • Track your learning progress
  • Generate statistics and metrics
  • Improve your learning experience

Data Storage

All data is stored locally in your browser using Chrome's chrome.storage.local API. This means:

  • Data never leaves your device
  • Data is not synced across devices (unless you enable Chrome sync)
  • You can delete all data by uninstalling the extension

Third-Party Services

OpenAI API (Optional)

If you use the AI sentence generation feature, words are sent to OpenAI's API. This is an optional feature that requires your own API key. We are not responsible for OpenAI's data handling practices.

Your Rights

You have the right to:

  • Access your data (stored locally in Chrome)
  • Delete your data (uninstall the extension or clear Chrome storage)
  • Opt out of AI features (simply don't configure an API key)

Data Security

Since all data is stored locally on your device, security depends on your device's security. We recommend:

  • Keeping your browser and operating system updated
  • Using strong device passwords
  • Not sharing your device with unauthorized users

Children's Privacy

Our extension does not knowingly collect information from children under 13. All data is stored locally and not transmitted to us.
